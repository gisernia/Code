# Serie A – Classifica storica a punteggio

Script Python che calcola una classifica storica della Serie A (girone unico)
assegnando a ogni squadra N punti per il primo posto, N-1 per il secondo, ecc.,
dove N è il numero di squadre in quella stagione.

## Output
- serie_a_totali.csv → classifica aggregata
- serie_a_dettaglio_stagioni.csv → dettaglio per stagione

## Uso
```bash
python calcola_classifica_all.py
```

http://calcio-seriea.net/
